Virtual Acting Ensemble Workflow

Project workflow
Preparation
Pre-analysis
Defining project goals and requirements.
Deep dive: Clarify needs, risks, creative limits before production begins.
Team formation
Clarifying participants and roles.
Deep dive: Build a core team with clear roles for design, tech, acting, and management.
Design
Dramaturgy
Creating conceptual framework and story.
Deep dive: Scenario, dramaturgy, character arcs are detailed with input from all team members.
Storyboard
Visual flow and drawing of main scenes.
Deep dive: Flowcharts, visuals, and step-by-step interaction plans are created and validated.
Production
Prototyping
Experimental staging and technical tests.
Deep dive: First technical builds, user tests, and staging in VR environments.
Performance
Live application of conceptual structure.
Deep dive: Real-time digital acting with user/audience interaction, streamed or experienced in VR.
Evaluation
Feedback
Evaluation with team and audience.
Deep dive: After-action review, audience feedback, UX, tech analysis; new iterations planned.
Reporting
Documenting process and outcomes.
Deep dive: Final report with outcomes, learning, technical and creative documentation.
